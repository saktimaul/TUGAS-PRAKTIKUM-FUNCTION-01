## Konsep Function dalam C++

Function dalam C++ memiliki struktur dasar:

```cpp
tipe_return nama_function(parameter) {
    // body function
    return nilai;
}
```

### Jenis-jenis Function:
1. **Function tanpa return value (void)**
2. **Function dengan return value**
3. **Function dengan parameter**
4. **Function tanpa parameter**

## Materi yang Dipelajari

- Deklarasi dan definisi function
- Parameter function (pass by value, pass by reference)
- Return value
- Function overloading
- Scope variabel dalam function
